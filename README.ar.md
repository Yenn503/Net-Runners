## 🔍 ما هو Net-Runner؟

Net-Runner هو إطار عمل متعدد الوكلاء للفريق الأحمر يحوّل التعليمات باللغة الطبيعية إلى تقييمات أمنية منظمة. تتحدث مع **قائد المهمة** — وهو يفوض الاستطلاع والاستغلال وإعداد التقارير وكل شيء آخر إلى وكلاء متخصصين مخصصين.

<details>
<summary><strong>لماذا Net-Runner؟</strong></summary>

- **لغة طبيعية** — لا حاجة لحفظ الأوامر أو البنية؛ صِف ما تريد اختباره
- **تنسيق متعدد الوكلاء** — وكلاء متخصصون متوازيون، كل منهم بمعرفة عميقة بالأدوات
- **حواجز أمان مدمجة** — نقاط تحقق من النطاق تمنع الإجراءات خارج الحدود
- **الأدلة أولاً** — كل إجراء مسجل، كل اكتشاف قابل للتتبع
- **ذاكرة مستمرة** — استرجاع RAG عبر الجلسات، لكل وكيل ولكل مشروع
- **141 أداة متصلة** — من `nmap` إلى `BloodHound` إلى `Ghidra`، جاهزة للتنفيذ

</details>

## 🚀 البداية السريعة

```bash
bun install
bun run build
node dist/cli.mjs
```

ثم اكتب تعليمات باللغة الطبيعية:

```
قيّم https://target.example — ابدأ بالاستطلاع، ثم اختبر ثغرات الويب.
```

يقوم Net-Runner باكتشاف نيتك تلقائيًا، وتهيئة المهمة بإعدادات آمنة، ثم يبدأ العمل.

## 🤖 ربط مزود LLM

Net-Runner يعمل مع عدة مزودي LLM. اضبط متغيرات البيئة قبل التشغيل.

### باستخدام `ANTHROPIC_API_KEY`

```bash
export ANTHROPIC_API_KEY="sk-ant-..."
node dist/cli.mjs
```

### OpenAI

```bash
export OPENAI_API_KEY="sk-..."
export OPENAI_MODEL="gpt-4o"          # اختياري
node dist/cli.mjs
```

### Google Gemini

```bash
export GEMINI_API_KEY="AIza..."
export GEMINI_MODEL="gemini-2.5-pro"   # اختياري
node dist/cli.mjs
```

### Ollama (محلي)

```bash
ollama serve
ollama pull llama3.1:8b
export OPENAI_BASE_URL="http://localhost:11434/v1"
export OPENAI_MODEL="llama3.1:8b"
node dist/cli.mjs
```

### أي API متوافق مع OpenAI

يعمل مع LM Studio، vLLM، Together AI، Groq، Fireworks أو أي نقطة نهاية متوافقة:

```bash
export OPENAI_API_KEY="your-key"
export OPENAI_BASE_URL="https://your-provider.com/v1"
export OPENAI_MODEL="your-model-name"
node dist/cli.mjs
```

## ⚙️ كيف يعمل

| **1. الكشف** | تحليل نية التقييم، تحديد الأهداف |

| **4. التفويض** | توجيه العمل إلى الوكلاء المتخصصين |
| **5. الحماية** | نقطة تحقق قبل أي إجراء عالي التأثير |
| **6. التسجيل** | تسجيل الأدلة والنتائج في الوقت الفعلي |
| **7. التذكر** | حفظ المعرفة لكل وكيل ومشروع للجلسات المستقبلية |
| **8. التقرير** | إنتاج تقييم منظم مدعوم بالأدلة |
