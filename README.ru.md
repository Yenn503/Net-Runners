## Что такое Net-Runner?

Net-Runner — это мультиагентный фреймворк для Red Team, который превращает инструкции на естественном языке в структурированные оценки безопасности. Вы общаетесь с **Руководителем задания** — он делегирует разведку, эксплуатацию, отчётность и всё остальное специализированным агентам.

<details>
<summary><strong>Почему Net-Runner?</strong></summary>

- **Естественный язык** — не нужно запоминать флаги или синтаксис; опишите, что хотите протестировать
- **Мультиагентная оркестрация** — параллельные агенты-специалисты, каждый с глубокими знаниями инструментов
- **Встроенные ограничители** — контрольные точки области действия предотвращают выход за границы
- **Доказательства в первую очередь** — каждое действие регистрируется, каждая находка прослеживается
- **Постоянная память** — RAG-извлечение между сессиями, по агентам и проектам
- **141 подключённый инструмент** — от `nmap` до `BloodHound` и `Ghidra`, готовых к выполнению

</details>

## Как это работает

| Шаг | Что происходит |
|-----|---------------|
| **1. Обнаружение** | Анализ намерения оценки, идентификация целей |
| **2. Инициализация** | Создание папки `.netrunner/` с безопасными значениями по умолчанию |
| **3. Инъекция** | Прикрепление области, авторизации и ограничений к каждому ходу модели |
| **4. Делегирование** | Направление работы агентам-специалистам |
| **5. Защита** | Контрольная точка перед любым действием с высоким воздействием |
| **6. Запись** | Регистрация доказательств и находок в реальном времени |
| **7. Запоминание** | Сохранение знаний по агентам и проектам для будущих сессий |
| **8. Отчёт** | Генерация структурированного вывода оценки на основе доказательств |

## Агенты-специалисты

| Агент | Специализация |
|:------|:-------------|
| **Руководитель задания** | Оркестрация, маршрутизация рабочих процессов, координация |
| **Специалист по разведке** | Обнаружение сети, DNS, OSINT, перечисление поддоменов |
| **Веб-тестирование** | XSS, SQLi, SSRF, обход аутентификации, сканирование CMS |
| **Тестирование API** | GraphQL, JWT, IDOR, массовое присвоение, анализ схем |
| **Сетевое тестирование** | SMB, SSH, FTP, эксплуатация сервисов, анализ трафика |
| **Специалист по эксплуатации** | Генерация полезных нагрузок, выполнение PoC, контролируемая эксплуатация |
| **Повышение привилегий** | SUID, эксплойты ядра, злоупотребление токенами, побег из контейнеров |
| **Латеральное перемещение** | Повторное использование учётных данных, пивотинг, перенаправление портов |
| **Специалист по AD** | LDAP/Kerberos, злоупотребление доверием, ADCS, BloodHound |
| **Специалист по ретесту** | Воспроизведение находок, проверка исправлений |
| **Специалист по доказательствам** | Курирование артефактов, цепочка хранения, форензика |
| **Специалист по отчётам** | Классификация серьёзности, краткое изложение, рекомендации по исправлению |
